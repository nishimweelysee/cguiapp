========================================
STOCK MANAGEMENT SYSTEM
Standalone Windows Package
========================================

HOW TO USE:
-----------
1. Extract this ZIP file to any folder
2. Double-click "stock_management.exe"
3. That's it! No installation needed.

========================================
SYSTEM REQUIREMENTS:
-----------
- Windows 7 or later (64-bit)
- No additional software required

========================================
IMPORTANT:
-----------
- Keep all files together in the same folder
- Don't delete any .dll files
- Don't delete the share\ folder
- Don't move stock_management.exe without the other files

========================================
WHAT'S INCLUDED:
-----------
- stock_management.exe    (Main application)
- *.dll files             (Required libraries - don't delete!)
- share\ folder           (GTK themes - don't delete!)
- data\ folder            (Application data - created automatically)

========================================
TROUBLESHOOTING:
-----------

If you see "DLL not found" error:
- Make sure all files are extracted to the same folder
- Don't move the .exe to a different location

If the application doesn't start:
- Right-click stock_management.exe → Properties → Unblock (if available)
- Try running as Administrator
- Check Windows Event Viewer for details

========================================
SUPPORT:
-----------
If you encounter any issues, make sure:
1. All files are in the same folder
2. You're using Windows 7 or later (64-bit)
3. No antivirus is blocking the application
