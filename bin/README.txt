========================================
Stock Management System - Standalone Package
========================================

HOW TO RUN:
-----------
Simply double-click "stock_management.exe"

That's it! No installation needed.

========================================
WHAT'S INCLUDED:
-----------
- stock_management.exe     (Main application)
- *.dll files              (Required libraries)
- share\ folder            (GTK themes and data)
- data\ folder             (Application data - created automatically)

========================================
DISTRIBUTION:
-----------
To share this application:

1. Copy the ENTIRE "bin" folder to the target Windows machine
2. Keep all files together (don't separate the .exe from DLLs)
3. Double-click stock_management.exe to run

IMPORTANT:
- Keep all files in the same folder
- Don't delete any .dll files
- Don't delete the share\ folder

========================================
SYSTEM REQUIREMENTS:
-----------
- Windows 7 or later (64-bit)
- No additional software installation required

========================================
TROUBLESHOOTING:
-----------

If you see "DLL not found" error:
- Make sure all .dll files are in the same folder as stock_management.exe
- Don't move the .exe to a different location without the DLLs

If the application doesn't start:
- Check Windows Event Viewer for error details
- Make sure you're running Windows 7 or later
- Try running as Administrator
